# Format Rules

Follow exactly when producing any markdown output:

- No `##` or `###` headers. Use `**Bold Label**` on its own line for section labels.
- Use `-` for bullet points. One line per bullet, no sub-bullets.
- Paragraphs: 1-3 sentences max. Keep them tight.
- Tables: use `|` syntax. One sentence per cell. Keep cells short.
- No emoji. No horizontal rules between sections. No numbered lists unless order matters.
- No "here is" or "below is" preamble. Start with the first piece of content.
